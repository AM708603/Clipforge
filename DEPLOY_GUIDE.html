Open the Worker editor and paste our code

After deploying, click Edit Code. You'll see a code editor. Select all the existing code and delete it.

Then open the file worker/index.js from your GitHub repo, copy its entire contents, and paste it into the editor.

ℹThe worker/index.js file is in the clipforge folder you downloaded. Open it in any text editor (Notepad, VS Code) → Select All → Copy → Paste into Cloudflare editor.

Netlify
UPDATE FRONTEND URL & DEPLOY ON NETLIFY
1

Update your Worker URL in the frontend file

Open frontend/index.html in any text editor. Find this line near the bottom:

const API = "https://YOUR-WORKER.workers.dev";

Replace it with your actual Worker URL from Step 2:

const API = "https://clipforge-worker.YOUR-NAME.workers.dev";

Save the file.

2

Push to GitHub

In your terminal / GitHub Desktop, commit and push the updated file:

git add frontend/index.html git commit -m "Set Cloudflare Worker URL" git push
ℹIf you haven't pushed to GitHub yet: create a new repo on github.com named clipforge, then push all files there.
3

Deploy on Netlify (free, no card)

Go to netlify.com → sign up free → Add new site → Import from GitHub → select your clipforge repo.
